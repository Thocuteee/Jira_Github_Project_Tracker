# 🎓 Jira – GitHub Project Tracker (SWP391)

## 📌 Project Title

**A Supporting Tool for Requirements and Project Progress Management in Software Project Course (SEOOP) Using Jira and GitHub**

---

## 📖 Overview

In Software Engineering education, managing software requirements and tracking project progress are essential skills for students to develop professional development practices.

This system is designed to support students and lecturers in managing requirements, tasks, and project progress by integrating **Jira** for requirement/task management and **GitHub** for source code and commit tracking.

The tool provides automated generation of reports and Software Requirement Specification (SRS) documents to improve transparency and evaluation accuracy in team-based software projects.

---

## 🎯 Objectives

* Support requirement management for student software projects
* Track task assignments and progress within project teams
* Analyze GitHub commit activities to evaluate individual contributions
* Generate structured SRS documents automatically
* Provide progress reports for lecturers and team leaders
* Improve project monitoring and assessment efficiency

---

## 🚨 Problem Statement

Students often face difficulties when:

* Creating **Software Requirement Specification (SRS)** documents from scattered Jira requirements
* Tracking task assignments and execution status within team members
* Providing accurate reports about **individual contributions through GitHub commits**
* Presenting project progress clearly to lecturers

This system aims to solve these challenges by offering a **centralized tracking and reporting platform**.

## 👥 Primary Actors

* 👑 **Admin**
* 👨‍🏫 **Lecturer**

## ⚙️ Functional Requirements

### 👑 Admin

* Manage student groups
* Manage lecturers
* Assign lecturers to groups
* Configure Jira & GitHub integration

### 👨‍🏫 Lecturer

* Manage students in assigned groups
* View requirements and tasks
* View project progress reports
* View GitHub commit statistics

### 🧑‍💼 Team Leader

* Manage group requirements
* Manage tasks
* Assign tasks to members
* Monitor task progress
* View team commit summaries

### 👨‍💻 Team Member

* View assigned tasks
* Update task status
* Commit source code to GitHub
* View personal task statistics
* View personal commit statistics
